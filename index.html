My HTML Cheat Sheet

Headings

Headings are from H1 - H6. A page should only have one H1, because google's webcrawlers use them for SEO.

Paragaphs

Paragraphs are used to contain long forms of text, like what you are reading right now. They use `p` tags.

Lists

Unordered Lists

Unordered lists are marked with bullet points and go in `ul` tags. Each list item goes in an `li` tag.

Types of lists:

  • Ordered Lists (`ol`)
  • Unordered Lists (`ul`)

Ordered Lists

Ordered lists are numbered and go in `ol` tags. Each list item goes in an `li` tag.

Links

Links allow a user to click on some text, and it brings them to a different section of the page, or to a different site completely. Like this for example. Go to DuckDuckGo to search for more information

Setting `target="_blank"` will open the link in a new window. See MDN

Images

You can use the img tag to add images to your page.

Content Section Tags

Article element

Represents a self-contained contained composition in a document, page, application, or site, which is intened to be independently distrubited or reusable. Examples inclues: a forum post, a magazine or newspaper article, or a blog entry, a product card, a user-submitted coment, an interactive widget or gadget, or any other independent item or content.

Footer element

A footer typically contains information about the author of the section, copyright data or links to related documents.

Header element

Represents introductory content, typically a group of introductory or navagational aids. It may contain some heading elements but also a log, a search form, an author name, and other elements.

Nav element

Represents a section of a page whose purpose is to provide navigation links, either within the current document of to other documents. Common examples of navigation sections are menus, tables of contents, and indexes.
